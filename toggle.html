Toggle Switch

En este tutorial crearemos un Toggle Switch, el cual tiene la misma función que un checkbox pero es mucho más moderno y atractivo visualmente como el que tenemos a continuación.

Para empezar definiremos el html. Usamos las etiquetas label e input de tipo checkbox y un span que nos servirá para asignar los demás estilos css.

Luego de esto obtenemos un checkbox común como el siguiente:

Ahora asignaremos los estilos. Primero formateremos el label, ya que nos servirá como contenedor. Le asignaremos una posición relativa y asignamos el tamaño de ancho y alto que queremos que tenga nuestro switch.

Ocultamos el checkbox de html.

Formateamos la caja del interruptor sobre la cual se deslizará la perilla de control o slider, le asignamos un color de fondo, sus esquinas las redondeamos a 50px y ponemos que el tiempo de la transición será de 0.4 segundos.

Ahora pintamos la perilla de control o slider usando el selector before, le asignamos el tamaño que tendrá que debe de ser menor al del contenedor que creamos al inicio. Le asignamos un color, en este caso blanco, lo redondeamos al 50% e igualmente ponemos que la transición de este durará 0.4 segundos.

Cambiamos el color de fondo del contenedor cuando el checkbox esta activado.

Para finalizar creamos la transición. Deslizamos el slider a la derecha cuando el checkbox esta activado.
