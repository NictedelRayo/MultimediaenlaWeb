Acordeón Desplegable

En este tutorial crearemos un Acordeón el cual sirve para desplegar distinta información de forma ordenada, se animará las áreas de contenido al abrir y cerrar.

Para iniciar, crearemos el siguiente código html, usaremos casillas de verificación. Todo se envolverá en un contenedor con la clase ac-container. Para cada elemento, tendremos una casilla de verificación, una etiqueta y un article que es la sección de contenido de ese elemento:

Definimos el ancho del acordeón y lo centramos.

Ahora hacemos que las "label" aparezcan como botones para poder darles clic. Le agregamos sombras de cuadro para crear el borde, le agregamos un padding, un color a la letra y un tamaño a el título de cada sección.

Al pasar el mouse haremos que el fondo de la etiqueta cambie de color al igual que el color de la letra.

Ahora indicamos que cuando se haga clic en una etiqueta esta cambiará de color al igual que la letra.

Agregamos el simbolo "+" a la etiqueta para que se muestre cuando a esta no se le ha dado clic.

Añadimos el simbolo "-" para que aparezca cuando se le ha dado clic a la etiqueta.

Indicamos que no queremos que se muestre la casilla de verificación (input).

Ahora para el área del contenido le indicamos un color de fondo, tendrá una altura inicial de 0px y se ocultará cualquier desbordamiento. Agregaremos una transición para la altura y la sombra del cuadro. La transición que estamos agregando aquí actuará al cerrar el contenido.

Definimos otra transición para cuando una de las etiquetas es seleccionada.

Indicamos el tamaño de la letra que tendrá el contenido y le agregamos un padding.

Y por último le damos un alto a la sección donde se verá el contenido seleccionado.
