Menu Desplegable

Para empezar definiremos el html. Creamos una lista con los link que va a tener nuestro menú, como se muestra a continuación.

Ahora le signamos un margin y el ancho que tendrá el menú, así como el tipo de fuente de la letra.

Hacemos que se quiten las viñetas de las listas.

Asignamos el color de fondo que tendrá nuestro menú, así como el color de la letra y le agregamos un padding.

Ahora le indicamos que cuando se pase el cursor por encima que cambie el color de fondo y tambien asignamos left a float, para que el menú se vea de forma horizontal.

Por último, hacemos que el submenú se oculte y que se muestré cuando se pase el mouse por el menú principal.
